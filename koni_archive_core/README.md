# koni_archive_core

Core abstractions for the koni_archive ecosystem — byte sources, entry model, typed exceptions, checksums, and the format-detection registry.

Part of the [koni_archive](https://github.com/zenbaku/koni_archive) monorepo.

> **Status: pre-release** (0.x, git-only). The API stays 0.x with lockstep
> minor bumps until it stabilizes — see the repository's `ROADMAP.md`.
